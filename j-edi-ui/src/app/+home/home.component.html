J-EDI

J-EDI is an example project that leverages SaltStack as the event broker and "glues" together disparate, automated workflows that are usually run by human beings. This repo contains all Saltstack state files, scripts and configuration files to run these workflows together by starting each workflow based on a pre-defined event that is created by any number of other workflows. Each workflow, in turn, creates an event so that if workflows need to start based on output from another workflow, they can do that automatically.
